Currently using a Acer Aspire V 13-371, in original configuration: Intel i3-4005U, 4 GB DDR3L and 500GB hard disk. This might look like an outdated device, but, surprisingly, it is still very usable. I will present now the it's capabilities and limits.

Operating sistem and core components

I am using Manjaro Linux and DWM window manager. This uses around 550 MB. Similar memory efficiency can be with XFCE desktop environment or I3 window manager

Web Browsing

Web browsing is preatty bloated, but it doesn't mean that it can't be done on this laptop. On LibreWolf I can open 9 tabs using 3.2 GB of RAM. It is a lot, but this is the web....

Android Sudio

I participate in the First Tech Challange robotics competition, and I need Android Studio to program the robot. It works, but can't open any other program.

I will update the list when I find more interesting and unexpected stuff I can do on this laptop!
